ABOUT PROJECT

This application/system helps to carry out promotional activities via the internet (website). This application has the main feature of providing reviews (Comments) of travel and adding travel photos. So that other users can see comments and photos brought by other users. There are three types of tourism available, namely natural tourism, artificial tourism, and cultural tourism.

This application is designed using CodeIgniter framework and assisted by HTML, CSS, and js. This website displays a specific tourist map, this map uses the leaflet.js. because leaflet.js is open source and free.
